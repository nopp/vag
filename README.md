Varnish Administration GUI 
=============================
(Open source VAC alternative)

* VCL Editor (vim like :))
* Purge

Obs:. I'm not a programmer :), do not expect a good code!

Required
========

* Python + Flask
* Varnish 3.x 4.x
* Varnish-agent 2.2+

To Do

How to use
==========

This tool works with secret file value.

python vag.py

http://ip:5010

* Admin user: admin/admin
* View user: view/view
